Documentación de Kitch API

Kitch es un API que provee la creación REST+JSON que posibilita el control de multiples aspectos de orden de comidas de para restaurantes y bares.

Login

Compruebe el Login utilizando: curl -i -H "Content-type: application/json" -X POST -d @request.txt http://localhost:5000/login/ Posibles Respuestas

User

Identifica al usuario del sistema, contiene datos del perfil. Requiere de un token de autorizacion para realizar llamadas a este recurso. Url: http://localhost:5000/user/ Posibles Respuestas

Menu

Requiere de un token de autorizacion para realizar llamadas a este recurso. Url: http://localhost:5000/menus/ Posibles Respuestas

MenuItems

Requiere de un token de autorizacion para realizar llamadas a este recurso. Url: http://localhost:5000/menuItems/ Posibles Respuestas
